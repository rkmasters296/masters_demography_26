# masters_demography_26
Supplementary material for simulating 2020 U.S. male life table

This repository provides supplementary material and examples for Masters's 2026 Demography article, "Research Note: 
A Method for Generating Accurate Provisional Life Tables for U.S. Populations." Repository contents:

1. "footnote 5.docx": Footnote #5, including Table S5 and Figure S7.
2. "us mortality_2019_2021.txt"
3. "mx ratio calculations.txt"
4. "Male 2020_2019 Life Table.xlsx"
5. "male_2020.txt"
6. "male_t_20.py"

Included in "footnote 5.docx" are descriptions of the files in #2–#6 above.
